# Fitness App README

## Overview

Welcome to our Fitness App, a cutting-edge Flutter application designed to empower users on their fitness journey. This comprehensive app offers a range of features, including a variety of home exercises, the ability to find nearby gyms, a BMI calculator for tracking your fitness progress, and a convenient platform for ordering health-conscious meals. With a sleek splash screen adorned with engaging Lottie animations, our app provides an immersive fitness experience. Below, you'll find an overview of the project, its dependencies, and screenshots showcasing the app's key features.

## Features

- **Diverse Home Exercises:** Achieve your fitness goals with a wide selection of home exercises tailored to different fitness levels and goals.

- **Find Nearby Gyms:** Discover and access nearby gyms effortlessly, enhancing your workout options.

- **BMI Calculator:** Monitor and evaluate your fitness journey with the integrated BMI calculator, offering valuable insights into your progress.

- **Healthy Diet Ordering:** Simplify your nutrition planning by ordering customized, health-conscious meals directly through the app.

- **Engaging Splash Screen:** Start your fitness journey with an engaging splash screen featuring captivating Lottie animations.

## Dependencies

Our Fitness App utilizes several dependencies to deliver a seamless user experience:

- **cupertino_icons (Version 1.0.2):** Provides Cupertino (iOS-style) icons to maintain a consistent and visually appealing interface.

- **provider (Version 6.0.3):** Implements efficient state management using the Provider pattern to facilitate data sharing throughout the app.

- **google_maps_flutter (Version 2.1.8):** Empowers users to locate nearby gyms effortlessly using Google Maps integration.

- **geolocator (Version 9.0.1):** Provides geolocation services to pinpoint users' locations for gym searches.

- **lottie (Version 1.3.0):** Enhances the app's splash screen with captivating Lottie animations.

- **http (Version 0.13.4):** Enables seamless communication with external APIs for various app functionalities.

- **fab_circular_menu (Version 1.0.2):** Implements a circular menu for intuitive navigation and enhanced user interaction.

## Getting Started

To experience the Fitness App firsthand, follow these steps:

1. Clone the repository to your local machine:

   ```shell
   https://github.com/Abhishekverma657/Fitness_app.git
   ```

2. Navigate to the project directory:

   ```shell
   cd fitness-app
   ```

3. Install the app's dependencies using Flutter:

   ```shell
   flutter pub get
   ```

4. Run the app on an emulator or physical device:

   ```shell
   flutter run
   ```

You're now ready to embark on your fitness journey with our Fitness App. Enjoy a wealth of features designed to empower your wellness endeavors and enhance your fitness experience.

Achieve your fitness goals, stay healthy, and download the Fitness App today!
